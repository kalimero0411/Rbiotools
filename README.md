# Rbiotools
R scripts for bioinformatics

| Code number     | Genetic code      |
| ------------- | ------------- |
| Rupdate          |  Bash script to update/install all R packages        |
| RNAseq          |  Analyze RNA sequencing data        |
| ChIPSeq          |  Analyze MACS3 output of ChIP sequencing        |
| MethylSeq          |  Analyze Bismark output of Bisulfite sequencing        |
| WGCNA          |  Perform WGCNA clustering on RNAseq data        |
| dups          |  Create duplcate plot from FASTQ alignment        |
| Ks          |  Ks ridge plot from MCScanX output        |
| aov_plot          |  Create Analysis of Variance plots        |
| blocks_prob          |  Brassicaceae Block analysis        |
